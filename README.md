# Neuro-Dub 🎙️🤖

**Neuro-Dub** — это инструмент на базе искусственного интеллекта для автоматизированного перевода и дубляжа видеоконтента. Проект объединяет передовые технологии обработки звука и генерации речи, чтобы превратить оригинальную аудиодорожку фильма или сериала в качественный дубляж на другом языке.

## ✨ Основные возможности

* **Audio Separation:** Отделение голоса от фоновой музыки и шумов с помощью **Demucs**.
* **Speech-to-Text:** Высокоточное распознавание речи через **Faster-Whisper**.
* **Translation & Adaptation:** Перевод текста с сохранением смысла и таймингов с помощью **GPT API**.
* **Text-to-Speech:** Генерация естественной озвучки (в процессе разработки).
* **Video Processing:** Сборка финального видео с новой дорожкой через **FFmpeg**.

## 🛠 Технологический стек

| Технология | Назначение |
| :--- | :--- |
| **Python** | Основной язык разработки |
| **Demucs** | Разделение аудио на вокал и инструментал |
| **Faster-Whisper** | Быстрая и точная транскрибация речи |
| **OpenAI GPT API** | Смысловой перевод и лингвистическая адаптация |
| **FFmpeg** | Манипуляции с видео и аудио потоками |

## 🚀 Установка и запуск

### Предварительные требования
Убедитесь, что у вас установлены **Python 3.10+** и **FFmpeg**.

1. **Клонируйте репозиторий:**
```bash

cd neuro-dub
python -m venv .venv
source .venv/bin/activate  
pip install -r requirements.txt
